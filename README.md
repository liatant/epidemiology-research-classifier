# Epidemiology Research Aim Classifier Web App

A tiny static web app for classifying epidemiologic research questions as:

- Descriptive
- Analytic
- Predictive
- Prescriptive

This app runs entirely in the browser. No server, database, login, or API key is required.

## Open locally

Double-click `index.html`.

## Share with friends using Netlify Drop

1. Go to Netlify Drop: https://app.netlify.com/drop
2. Drag the entire `epi_aim_web_app` folder onto the page.
3. Netlify will give you a public link.
4. Send that link to friends.

## Share with friends using GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `app.js`.
3. Go to Settings > Pages.
4. Choose the main branch and root folder.
5. GitHub will give you a public link.

## Files

- `index.html`: page structure
- `styles.css`: visual design
- `app.js`: browser-based classifier logic

## Notes

This is a rule-based educational classifier. It is useful for triage and teaching, but borderline research questions should be reviewed by an epidemiologist.
